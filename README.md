# Learn Apache Iceberg & Spark

Build a practical data pipeline while learning how Apache Iceberg and Apache Spark work together.

This repository contains the companion code for **Ollie's Growing Tips: Apache Iceberg & Spark 101**, a free, hands-on course from [oleander](https://oleander.dev).

## [Start the free tutorial →](https://oleander.dev/grow)

The tutorial takes you from the fundamentals of storage and compute to building real Spark and Iceberg workloads. Along the way, you will learn how to:

- Model and transform retail clickstream data
- Work with Iceberg tables using Spark SQL and the DataFrame API
- Build aggregations, joins, and retry-safe pipelines
- Improve query performance with partitioning and time travel
- Stream events into Iceberg and compact small files

No previous Iceberg experience is required. The course provides the explanations, setup, sample data, and step-by-step instructions.

## About this repository

The directories in this repository correspond to the later, code-heavy parts of the course:

- Parts 8–12 include equivalent Spark SQL and DataFrame implementations
- Part 15 streams Kafka events into an Iceberg table
- Part 16 compacts small Iceberg data files

These projects are designed to be used alongside the tutorial rather than as standalone examples. Open the relevant lesson, then use its matching directory here when you are ready to run or explore the complete code.
